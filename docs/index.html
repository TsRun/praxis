OpeningTree

Real OTB games,
at the move you actually play. Now.

A chess opening explorer built on a deduped, FIDE-id-linked database of over-the-board master games. Pick a move from a dense, visual list of candidates — each row gives you a games count, the average Elo of the people who played it, and the W/D/B split, with a position-preview on hover.

What's in it

Real OTB games

2.35M deduped over-the-board master games today. Same pipeline scales to 10M+ from Lumbra's Gigabase or TWIC's full archive.

FIDE player filter

Autocomplete over 1.84M FIDE-listed players. Filter the explorer to a player's games as any, white, or black.

Move-by-move stats

For every candidate move at the current position: total games, average Elo of the players who chose it, W/D/B distribution.

Fast at scale

Precomputed move_stats table for the unfiltered path — millions of games, single-digit-millisecond queries.

Keyboard nav

← / → step through the line, Home rewinds, End jumps to the latest played ply.

State survives reload

Current line + ply pointer persist via localStorage — no URL hacks, refreshing doesn't lose your place.

Run it

One-time setup

setup
# Postgres 16 (Homebrew on macOS)
$ brew install postgresql@16
$ brew services start postgresql@16
$ createdb openings

# Project deps
$ npm install

Daily

dev
$ npm run dev:server    # Fastify on :5174
$ npm run dev           # Vite on :5173

Filling the database

data
$ npm run load-fide                        # ~1.84M players
$ npm run ingest -- --twic 920..1614       # latest 13 years of OTB
$ npm run ingest -- --file path/to.pgn     # or any PGN file
$ npm run rebuild-stats                    # auto-run by ingest unless --no-stats

Stack

Vite · React 18 · TypeScript · Tailwind · Zustand · chess.js · chessground · Fastify · node-postgres · sax-js · Postgres 16 with pg_trgm · Vitest · Playwright.
