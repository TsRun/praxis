Smallest-change fix: switch the mobile grid rule to `minmax(0, 1fr)`, add `flexWrap: 'wrap'` plus a `flex: 1 1 200px; minWidth: 0` wrapper to both `PlayFromStartBoard` and `PositionSetupBoard` so the toolbar drops below the board on narrow viewports, and add `wordBreak: 'break-all'` to the Filter FEN line.
